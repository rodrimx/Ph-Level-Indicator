# Ph-Level-Indicator
//Very easy and simple JavaScript coded Ph Level indicator that determines whether the level is acidic, basic, or neutral.
#Codedex

Const ph = 3;

if (ph > 7) {
    console.log ("Acidic")
} else if (ph < 7) {
    console.log ("Basic")
} else {
    console.log ("Neutral")
}

//Output should be either acidic, basic, or neutral, depending on the value given to the ph level at the start of the code.
